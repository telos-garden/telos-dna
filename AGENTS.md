# AGENTS

Este repositório (TELOS-DNA) aceita contribuições humanas e assistidas por agentes de IA, desde que todas respeitem os princípios conceituais, filosóficos e estruturais aqui definidos.

Este arquivo estabelece o contrato operacional para qualquer agente (humano ou artificial) que atue neste repositório.

⸻

Propósito do repositório

TELOS-DNA é o repositório de governança conceitual do ecossistema TELOS.

Ele contém:
 • fundamentos filosóficos
 • conceitos operacionais
 • princípios de documentação
 • regras de coerência e rastreabilidade de sentido

Não contém:
 • código de produto
 • implementações específicas
 • decisões locais de projetos derivados

Todo conteúdo aqui é considerado material genético conceitual.

⸻

Princípios obrigatórios para agentes

Qualquer agente que produza, modifique ou revise conteúdo neste repositório DEVE respeitar os princípios abaixo.

1. Autonomia dos documentos
 • Cada documento deve ser compreensível de forma isolada.
 • Nenhum documento deve assumir leitura prévia de outro.
 • Referências cruzadas devem ser explícitas, nunca implícitas.

2. Significado relacional
 • Nenhum documento é fonte de verdade isolada.
 • Todo significado relevante emerge da relação declarada entre documentos.
 • Alterações devem preservar ou esclarecer essas relações.

3. Rastreabilidade de decisões
 • Toda decisão conceitual deve indicar seus fundamentos.
 • Nenhuma alteração estrutural pode ser feita sem justificar o porquê, não apenas o como.

4. Não instrumentalização da filosofia
 • Filosofia não deve ser usada como retórica decorativa.
 • Conceitos filosóficos só podem ser citados quando forem relevantes para a decisão em questão.

5. Neutralidade de implementação
 • Este repositório não favorece linguagens, frameworks ou stacks.
 • Exemplos técnicos, quando existirem, devem ser ilustrativos, não prescritivos.

⸻

Escopo permitido para agentes

Agentes podem:
 • redigir novos documentos filosóficos
 • propor novos conceitos operacionais
 • revisar clareza, consistência e estrutura de documentos existentes
 • melhorar rastreabilidade entre textos

Agentes não podem:
 • introduzir decisões de produto
 • impor padrões técnicos específicos
 • reduzir conceitos para simplificações utilitárias
 • apagar tensões conceituais sem registro explícito

⸻

Regras para contribuições assistidas por IA

Contribuições feitas com auxílio de IA devem atender aos critérios abaixo:
 • o conteúdo deve ser revisado conscientemente por um humano responsável
 • a IA não deve ser tratada como autoridade conceitual
 • ambiguidades devem ser preservadas quando forem reais, não eliminadas artificialmente

IA é ferramenta de clareza, não fonte de verdade.

⸻

Estilo e forma

Todo conteúdo neste repositório deve:
 • seguir Markdown compatível com markdownlint
 • evitar listas excessivamente profundas
 • usar linguagem clara, direta e não promocional
 • evitar slogans, marketing ou tom persuasivo

Clareza tem prioridade sobre elegância.

⸻

Governança implícita

Este arquivo não substitui documentos de governança.

Ele existe para garantir que qualquer agente saiba onde está pisando antes de agir.

Se um agente não consegue operar dentro destas restrições, ele não deve atuar neste repositório.

⸻

Princípio final

TELOS-DNA não existe para convencer.

Existe para permanecer coerente ao longo do tempo, mesmo quando interpretado, estendido ou questionado.

Agentes são bem-vindos.

Mas o jardim vem primeiro.
